    환급신청 시 수강료가 현금 환급됩니다.

  2. 2. 신청절차

    최종합격 후 7일 이내 지정된 카페 또는 개인 블로그에 합격수기 작성 후
    에듀윌 사이트 합격 수기에 내용/URL/ID 등록 후 환급 신청을 하시면 됩니다.
    ① 지정 카페 또는 개인 블로그에 합격수기 등록 (카페,블로그는 환급신청 전 추후 공지)
    ② 에듀윌 사이트 시험후기/합격수기 게시판에 합격수기, URL, 카페 ID 등록
    ③ 공지사항에 게시된 안내에 따라 환급 신청

  3. 3. 환급액 산정

    1) 교재 판매가, 제세공과금 제외 없이 결제 금액 전액 환급됩니다.
    2) 환급은 하단의 '유의사항'에 있는 조건들을 충족하신 경우에만 적용됩니다.

  4. 4. 유의사항

    1) 시험 미 응시로 수강기간 종료 시 환급 불가합니다.
    2) 최종 합격자 발표 후 7일 이내 신청하지 않으면 환급이 지급되지 않습니다.
    3) 합격 인증 자료는 ‘평생환급 평생패스’ 구매 아이디/합격증/본인 신분증 사본의 이름(실명)이 동일할 경우에 적용됩니다.
    4) 소정의 합격수기를 ‘합격수기’ 게시판에 등록하셔야 합니다.(최종 합격자 발표 후 각 7일 이내)
    5) 수강료 환급 신청 접수 후에는 잔여 수강 기간에 관계없이 수강 기간이 종료됩니다.
